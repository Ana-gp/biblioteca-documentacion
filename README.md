# 📚 Sistema de Gestión Bibliotecaria

## Descripción
Sistema para gestionar préstamos de libros en una biblioteca pública.

## Características
- ✅ Catálogo de libros
- ✅ Gestión de usuarios
- ✅ Control de préstamos y devoluciones
- ✅ Cálculo de multas por retraso
